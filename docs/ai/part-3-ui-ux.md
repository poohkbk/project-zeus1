# 1. UX 설계 철학

제우 AI는

일반적인 챗봇과 다르다.

예를 들어

일반 AI는

```
질문

↓

긴 답변
```

으로 끝난다.

하지만

제우 AI는

```
질문

↓

↓

추가 질문

↓

관련 승소사례

↓

관련 법률가이드

↓

FAQ

↓

상담예약
```

으로 끝난다.

즉,

**답변보다 상담 연결이 목표**이다.

# 16. UX 설계 원칙

제우 AI는 **"많이 말하는 AI"가 아니라 "필요한 질문만 하고, 필요한 자료를 안내하며, 적절한 콘텐츠와 상담으로 연결하는 AI"**를 목표로 합니다.

이를 위해 모든 사건 분야(민사·형사·이혼·상속·행정)에 동일한 UX 패턴을 적용합니다.

```text
질문
↓
사건분류
↓
추가 질문
↓
준비자료 안내
↓
관련 승소사례
↓
관련 법률가이드
↓
FAQ
↓
상담예약
```

이 구조는 사용자가 **설명서를 읽지 않아도 자연스럽게 다음 단계로 이동**할 수 있도록 설계되어 있으며, 콘텐츠가 늘어날수록 AI의 추천 품질만 향상되고 화면과 사용 방식은 그대로 유지되는 확장형 UX입니다.
